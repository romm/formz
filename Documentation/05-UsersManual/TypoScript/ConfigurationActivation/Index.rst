.. _conditionType:

Condition type
--------------

.. container:: table-row

    Property
        ``activation``
    Required?
        No
    Description
        Contains the condition type. You can find all the condition types available below.

        **Example:**

        .. code-block:: typoscript

            activation {
                items {
                    emailIsValid {
                        type = fieldIsValid
                        fieldName = email
                    }
                }
            }
